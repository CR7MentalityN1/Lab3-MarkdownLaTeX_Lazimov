## Лабораторная работа №3. Работа с Markdown-разметкой и базовое использование LaTeX в документации проекта

### Что такое Markdown

Markdown — это лёгкий язык разметки, который используется для оформления README-файлов, документации, отчётов и даже научных текстов. Он прост в использовании, хорошо читается как в сыром виде, так и после рендеринга, и широко поддерживается GitHub, VS Code и многими платформами.

### Где применяется Markdown

Markdown находит широкое применение в веб-разработке, блогах, документации, системах управления версиями, заметках и презентациях. Его используют GitHub, GitLab, многие редакторы кода и платформы для написания текстов.

### Зачем Markdown нужен разработчикам

Разработчикам Markdown нужен для быстрого и простого написания документации, README-файлов, заметок, комментариев и сообщений, которые легко читать и форматировать. Он экономит время, повышает читательность и обеспечивает единообразное оформление текста.
