# Примеры горизонтальных линий в Markdown

---

## Введение

Горизонтальные линии используются для разделения крупных смысловых блоков в документе. В Markdown их можно создавать с помощью трех и более символов: дефисов `---`, звездочек `***` или нижних подчеркиваний `___`.

---

## Основные возможности Markdown

Markdown предоставляет простые и удобные средства форматирования текста:

- **Заголовки** разных уровней
- **Списки** (маркированные и нумерованные)
- **Выделение текста** (жирный, курсив, зачеркнутый)
- **Ссылки и изображения**
- **Таблицы**
- **Цитаты и блоки кода**
