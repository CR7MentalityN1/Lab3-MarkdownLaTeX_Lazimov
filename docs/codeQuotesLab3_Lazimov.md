### Блоки кода с подсветкой синтаксиса

**Python:**

```python
def hello(name):
    print(f"Hello, {name}!")

hello("Markdown")

```

**Js:**

```Js
function hello(name) {
    console.log(`Hello, ${name}!`);
}

hello("Markdown");
```
