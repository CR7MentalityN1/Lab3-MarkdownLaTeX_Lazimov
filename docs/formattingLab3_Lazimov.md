# Примеры текстового форматирования в Markdown

## Базовое форматирование

| Формат                  | Синтаксис                   | Пример                | Результат             |
| ----------------------- | --------------------------- | --------------------- | --------------------- |
| Жирный текст            | `**текст**` или `__текст__` | **жирный текст**      | **жирный текст**      |
| Курсив                  | `*текст*` или `_текст_`     | _курсив_              | _курсив_              |
| Жирный курсив           | `***текст***`               | **_жирный курсив_**   | **_жирный курсив_**   |
| Зачёркнутый             | `~~текст~~`                 | ~~зачёркнуто~~        | ~~зачёркнуто~~        |
| Моноширный (inline код) | `код`                       | `Console.WriteLine()` | `Console.WriteLine()` |

---

## Блоки кода

### Блок кода без указания языка:

```
{
"firstName": "Иван",
"lastName": "Петров",
"age": 25
}
```

### Блок кода с указанием языка (C#):

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, Markdown!");

        string name = "Иван";
        int age = 25;

        Console.WriteLine($"Имя: {name}, Возраст: {age}");
    }
}

### Блок кода с указанием языка (JavaScript):
function greet(name) {
    return `Привет, ${name}!`;
}

console.log(greet('Мир'));


### Блок кода с указанием языка (bash):
git init
git add .
git commit -m "Initial commit"
git push -u origin main
```
