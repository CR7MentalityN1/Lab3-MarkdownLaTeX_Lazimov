### Форматирование текста

- **Полужирный текст** — для важных элементов
- _Курсив_ — для определений и пояснений
- **_Жирный курсив_** — для особо важных терминов
- ~~Зачёркнутый текст~~ — для устаревшей информации
- `Моноширный текст` — для команд и кода

---

### Списки

#### Маркированный список

- Пункт 1
- Пункт 2
- Пункт 3

#### Нумерованный список

1. Первый пункт
2. Второй пункт
3. Третий пункт

#### Вложенный список

1. Основной пункт
   - Подпункт A
   - Подпункт B
     1. Подподпункт 1
     2. Подподпункт 2
2. Другой пункт

---

### Цитата

> Markdown — это легкий язык разметки, созданный для удобного написания документации.
> Он позволяет быстро форматировать текст без использования сложных HTML-тегов.

---

### Блок кода (с указанием языка C#)

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, Markdown!");

        // Пример переменных
        string name = "Марк";
        int age = 25;

        Console.WriteLine($"Имя: {name}, Возраст: {age}");
    }
}

### Таблица
| Формат                  | Синтаксис                   | Пример                | Результат             |
| ----------------------- | --------------------------- | --------------------- | --------------------- |
| Жирный текст            | `**текст**` или `__текст__` | **жирный текст**      | **жирный текст**      |
| Курсив                  | `*текст*` или `_текст_`     | _курсив_              | _курсив_              |
| Жирный курсив           | `***текст***`               | **_жирный курсив_**   | **_жирный курсив_**   |
| Зачёркнутый             | `~~текст~~`                 | ~~зачёркнуто~~        | ~~зачёркнуто~~        |
| Моноширный (inline код) | `код`                       | `Console.WriteLine()` | `Console.WriteLine()` |
```
